# Concurrency & Parallelism: zero-to-hero kitoblar

Bu yo'nalish thread, goroutine, lock, channel, memory model, parallel algorithm, lock-free data structure va multicore programming tafakkurini beradi. Avval inglizcha original kitoblar tanlandi, keyin ruscha tarjima/nashr holati tekshirildi.

## O'qish tartibi

1. **Concurrency in Go** bilan Go'dagi goroutine, channel, cancellation va pipeline modelini tushunish.
2. **Java Concurrency in Practice** yoki **C++ Concurrency in Action** bilan klassik thread, lock, memory model va race muammolariga kirish.
3. **Seven Concurrency Models in Seven Weeks** bilan actor, CSP, STM, data parallelism kabi turli yondashuvlarni solishtirish.
4. **Patterns for Parallel Programming** bilan parallel algoritmlarni loyihalash patternlarini ko'rish.
5. **The Art of Multiprocessor Programming** bilan lock-free, wait-free, consensus number va multicore theory darajasiga o'tish.
