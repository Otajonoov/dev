# System Design / Distributed Systems: zero-to-hero kitoblar

Bu ro'yxat system design interview asoslaridan boshlab scalability, distributed systems, microservices, reliability, SRE va distributed algorithms expert darajasigacha olib boradi. Avval inglizcha original kitoblar tanlandi, keyin ruscha tarjima/nashr holati tekshirildi.

## O'qish tartibi

1. **System Design Interview** bilan framework va interview style masalalarni o'rganish.
2. **Web Scalability for Startup Engineers** va **Designing Distributed Systems** bilan amaliy scalability/distributed patterns'ga o'tish.
3. **DDIA**, **Building Microservices** va **Release It!** bilan real production architecture tafakkurini qurish.
4. **SRE**, **Database Internals**, **Distributed Systems** va **Distributed Algorithms** bilan expert darajaga chiqish.
