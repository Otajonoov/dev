# Database: zero-to-hero kitoblar

Bu ro'yxat SQL asoslaridan boshlab DBMS nazariyasi, internals, transactions va distributed database darajasigacha olib boradi. Avval inglizcha original kitoblar tanlandi, keyin ruscha tarjima/nashr holati tekshirildi.

## O'qish tartibi

1. **Learning SQL** va **SQL Queries for Mere Mortals** bilan query yozishni o'rganish.
2. **C. J. Date** yoki **Database Systems: The Complete Book** bilan DB nazariyasini mustahkamlash.
3. **DDIA** va **Database Internals** bilan zamonaviy storage/distributed data systems ichiga kirish.
4. **Transaction Processing**, **Principles of Distributed Database Systems** va **Red Book** bilan expert/research darajasiga o'tish.
