# Computer Networking: zero-to-hero kitoblar

Bu ro'yxat network asoslaridan boshlab TCP/IP internals, web networking performance va routing expert darajasigacha olib boradi. Avval inglizcha original kitoblar tanlandi, keyin ruscha tarjima/nashr holati tekshirildi.

## O'qish tartibi

1. **Computer Networking: A Top-Down Approach** bilan application layerdan boshlab tarmoqlarni tushunish.
2. **Computer Networks** yoki **Data and Computer Communications** bilan fundamental modelni mustahkamlash.
3. **Computer Networks: A Systems Approach** va **Internetworking with TCP/IP** bilan systems/TCP-IP tafakkurga o'tish.
4. **TCP/IP Illustrated**, **High Performance Browser Networking** va **Routing TCP/IP** bilan chuqurlashish.
