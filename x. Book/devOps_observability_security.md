# DevOps / Observability / Security: zero-to-hero kitoblar

Bu yo'nalish production muhandislikni yopadi: Linux administration, CI/CD, DevOps culture, deployment automation, Kubernetes, SRE, observability, incident response va security. Avval inglizcha original kitoblar tanlandi, keyin ruscha tarjima/nashr holati tekshirildi.

## O'qish tartibi

1. **UNIX and Linux System Administration Handbook** bilan production Linux/SRE poydevorini qurish.
2. **The Phoenix Project**, **The DevOps Handbook** va **Accelerate** bilan DevOps madaniyati, flow, feedback va DORA metrikalarini tushunish.
3. **Continuous Delivery** va **Kubernetes in Action** bilan deployment pipeline va container orchestration amaliyotiga o'tish.
4. **Site Reliability Engineering**, **Observability Engineering** va **Distributed Systems Observability** bilan monitoringdan haqiqiy observability'ga o'tish.
5. **Web Application Security**, **Real-World Cryptography** va **Security Engineering** bilan security thinking va secure systems design'ni mustahkamlash.
