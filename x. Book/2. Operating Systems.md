# Operating Systems: zero-to-hero kitoblar

Bu ro'yxat OS'ga tayanch systems bilimdan boshlab virtualization, concurrency, persistence, Linux API va kernel internals darajasigacha olib boradi. Avval inglizcha original kitoblar tanlandi, keyin ruscha tarjima/nashr holati tekshirildi.

## O'qish tartibi

1. **Computer Systems: A Programmer's Perspective** bilan systems foundation olish.
2. **Operating Systems: Three Easy Pieces** bilan OS'ning 3 katta blokini tushunish.
3. **Operating System Concepts**, **Modern Operating Systems** va **Stallings** bilan textbook darajasida mustahkamlash.
4. **OS Design and Implementation**, **The Linux Programming Interface** va **Understanding the Linux Kernel** bilan amaliy/expert chuqurlikka o'tish.
