# Go / Golang: zero-to-hero kitoblar

Bu ro'yxat Go'ni noldan boshlab production, concurrency va performance darajasigacha olib borish uchun tuzildi. Avval inglizcha original kitoblar tanlandi, keyin ruscha tarjima/nashr holati tekshirildi.

## O'qish tartibi

1. **Head First Go** yoki **The Go Programming Language** bilan til asoslarini o'rganish.
2. **Learning Go** va **Go in Practice** bilan idiomatic Go va production usullarni mustahkamlash.
3. **Let's Go!** bilan real web backend qurish.
4. **Concurrency in Go**, **100 Go Mistakes** va **Efficient Go** bilan senior darajaga chiqish.
