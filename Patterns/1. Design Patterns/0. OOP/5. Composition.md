# Kompozitsiya (Composition)

## Umumiy tushuntirish

**Kompozitsiya** — agregatsiyaning **qat'iy** turi. Bir ob'ekt boshqa ob'ektlardan iborat, va bo'lak **faqat konteyner tarkibida** mavjud bo'la oladi.

```
Uy ◆──────> Xona
   "butun-bo'lak"

◆ — to'ldirilgan romb = kompozitsiya belgisi (UML da)
```

**Mohiyati:** Konteyner yo'qolsa, bo'laklar **ham yo'qoladi**.

```
Kompozitsiya:
  ❌ Uy yo'qolsa → Xonalar ham yo'qoladi
  ❌ Odam yo'qolsa → Yurak ham yo'qoladi
  ❌ Buyurtma o'chirilsa → Buyurtma elementlari ham o'chiriladi
```

**Agregatsiya vs Kompozitsiya:**

| | Agregatsiya ◇ | Kompozitsiya ◆ |
|-|---------------|----------------|
| Bo'laklar yaratilishi | Tashqarida | Ichida |
| Bo'laklar mustaqilligi | ✅ Mustaqil | ❌ Mustaqil emas |
| Konteyner o'chirilsa | Bo'laklar qoladi | Bo'laklar ham o'chadi |
| Egalik | Zaif | Kuchli |
| Misol | Sinf → O'qituvchi | Uy → Xona |

## Python

```python
# Kompozitsiya — bo'laklar konteyner ICHIDA yaratiladi
class Xona:
    def __init__(self, nomi: str, maydoni: float):
        self.nomi = nomi
        self.maydoni = maydoni

    def __repr__(self) -> str:
        return f"Xona({self.nomi}, {self.maydoni}m²)"

    def __del__(self):
        print(f"  '{self.nomi}' xonasi yo'qoldi")


class Uy:
    def __init__(self, manzil: str, xona_soni: int):
        self.manzil = manzil
        # Xonalar UY ICHIDA yaratiladi — kompozitsiya
        self.__xonalar = [
            Xona("Yotoqxona", 18.0),
            Xona("Oshxona", 12.0),
            Xona("Zal", 25.0),
        ][:xona_soni]

    @property
    def xonalar(self) -> list[Xona]:
        return self.__xonalar

    def umumiy_maydon(self) -> float:
        return sum(x.maydoni for x in self.__xonalar)

    def tavsif(self) -> str:
        return f"Uy [{self.manzil}]: {len(self.__xonalar)} xona, {self.umumiy_maydon()}m²"

    def __del__(self):
        print(f"\nUy '{self.manzil}' buzildi — barcha xonalar yo'qoladi:")


# Foydalanish
uy = Uy("Navoiy ko'chasi, 15", 3)
print(uy.tavsif())
print(uy.xonalar)

# Uy yo'qolganda xonalar ham yo'qoladi
del uy
```

**Natija:**
```
Uy [Navoiy ko'chasi, 15]: 3 xona, 55.0m²
[Xona(Yotoqxona, 18.0m²), Xona(Oshxona, 12.0m²), Xona(Zal, 25.0m²)]

Uy 'Navoiy ko'chasi, 15' buzildi — barcha xonalar yo'qoladi:
  'Yotoqxona' xonasi yo'qoldi
  'Oshxona' xonasi yo'qoldi
  'Zal' xonasi yo'qoldi
```

## Go

```go
package main

import "fmt"

// Kompozitsiya — bo'laklar value type sifatida ichida saqlanadi
type Xona struct {
    Nomi    string
    Maydoni float64
}

func (x Xona) String() string {
    return fmt.Sprintf("%s (%.1fm²)", x.Nomi, x.Maydoni)
}

type Uy struct {
    Manzil string
    xonalar []Xona  // private, value type — Uy bilan birga
}

func YangiUy(manzil string) *Uy {
    return &Uy{
        Manzil: manzil,
        // Xonalar UY ICHIDA yaratiladi
        xonalar: []Xona{
            {Nomi: "Yotoqxona", Maydoni: 18.0},
            {Nomi: "Oshxona", Maydoni: 12.0},
            {Nomi: "Zal", Maydoni: 25.0},
        },
    }
}

func (u *Uy) UmumiyMaydon() float64 {
    var jami float64
    for _, x := range u.xonalar {
        jami += x.Maydoni
    }
    return jami
}

func (u *Uy) Tavsif() string {
    return fmt.Sprintf("Uy [%s]: %d xona, %.1fm²",
        u.Manzil, len(u.xonalar), u.UmumiyMaydon())
}

func main() {
    uy := YangiUy("Navoiy ko'chasi, 15")
    fmt.Println(uy.Tavsif())

    // uy.xonalar → faqat Uy ichidan ko'rinadi
    // Uy nil bo'lganda xonalar ham yo'qoladi (value type)
}
```
