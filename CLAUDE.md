# CLAUDE

Bu repo Go dasturlash tilini o'rganish uchun ishlatiladi.

## Agents

| Agent           | Vazifasi                                           |
| --------------- | -------------------------------------------------- |
| **`teacher`**   | Tarjima, tushuntirish, diagrammalar, md ga saqlash |
| **`quiz-master`** | Test tuzish, bilimni tekshirish                  |

## Skills

| Skill         | Ishlatish                    | Vazifasi                                      |
| ------------- | ---------------------------- | --------------------------------------------- |
| **`/learn`**  | `/learn golang concurrency`  | Matnni tarjima + tushuntirish + faylga saqlash |
| **`/quiz`**   | `/quiz golang`               | Mavzu bo'yicha test tuzish                     |
| **`/explain`**| `/explain mutex`             | Biror tushunchani chuqur tushuntirish          |
