# 11 — Amaliyot va Interview

Bu modul — Network kursining **yakuniy amaliy qismi**. Oldingi 11 modulda o'rganilgan nazariyani (00-tarmoq-asoslari dan 10-go-network-programming gacha) amalda mustahkamlaydi va intervyuga tayyorlaydi.

Modul uch yo'nalishda ishlaydi: **qur** (lab), **tuzat** (troubleshooting), **javob ber** (interview).

## Fayllar

| Fayl | Mazmun |
|------|--------|
| [01-lab-topologies.md](./01-lab-topologies.md) | 12 ta Packet Tracer / GNS3 lab: topologiya, konfiguratsiya, tekshirish buyruqlari, kutilgan natija |
| [02-troubleshooting-cases.md](./02-troubleshooting-cases.md) | Layer-by-layer metodologiya + 17 real case (symptom -> diagnostika -> root cause -> yechim) |
| [03-interview-questions.md](./03-interview-questions.md) | 90+ intervyu savoli, mavzu bo'yicha guruhlangan, javoblar yashiringan |
| [04-coverage-checklist.md](./04-coverage-checklist.md) | 12 modul bo'yicha bilim checklisti (checkbox) |

## Foydalanish tartibi

1. **Checklist bilan boshla** — 04-coverage-checklist.md dan qaysi mavzuni bilishing kerakligini ko'r.
2. **Lab qil** — har modulni o'qigach, 01-lab-topologies.md dagi tegishli labni o'z qo'ling bilan yig'.
3. **Buzganda tuzat** — lab ishlamasa yoki real muammoda 02-troubleshooting-cases.md dagi metodologiyani qo'lla.
4. **Intervyuga tayyorlan** — 03-interview-questions.md savollariga avval o'zing javob ber, keyin tekshir.

> **Maslahat:** Bu modul bir marta o'qib qo'yiladigan material emas. Uni **takrorlash oynasi** sifatida ishlat — ertaga, 3 kundan keyin, 1 haftadan keyin qaytib kel.
