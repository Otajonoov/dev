# Network, broadcast va host range

## Subnet ichidagi 3 asosiy address tushunchasi

Har subnetda odatda quyidagi tushunchalar bor:

1. **Network address** - subnetning nomi. Host bitlar hammasi `0`.
2. **Broadcast address** - subnetdagi hammaga yuborish addressi. Host bitlar hammasi `1`.
3. **Usable host range** - qurilmalarga beriladigan addresslar.

Masalan:

```text
192.168.1.0/24
```

Bu subnetda:

```text
Network address:   192.168.1.0
First usable host: 192.168.1.1
Last usable host:  192.168.1.254
Broadcast address: 192.168.1.255
Usable hosts:      254 ta
```

## Network address

Network address subnetning identifikatori.

`192.168.48.68/24` uchun:

```text
IP:    192.168.48.68
Mask:  255.255.255.0
```

Host qismi oxirgi octet. Host bitlarni `0` qilsak:

```text
Network address: 192.168.48.0
```

Network address odatda hostga berilmaydi.

## Broadcast address

Broadcast address subnet ichidagi barcha hostlarga packet yuborish uchun ishlatiladi.

`192.168.48.68/24` uchun host bitlarni `1` qilsak:

```text
Broadcast address: 192.168.48.255
```

Broadcast address ham odatda hostga berilmaydi.

## Host range

Usable hostlar network va broadcast address orasida joylashadi.

```text
Network:   192.168.48.0
Broadcast: 192.168.48.255
Hosts:     192.168.48.1 - 192.168.48.254
```

## Block size

Subnet range'ni tez topish uchun block size ishlatiladi:

```text
Block size = 256 - qiziq octetdagi mask qiymati
```

Masalan:

```text
192.168.1.64/26
```

`/26` mask:

```text
255.255.255.192
```

Qiziq octet oxirgisi:

```text
256 - 192 = 64
```

Demak subnetlar 64 qadam bilan yuradi:

```text
192.168.1.0
192.168.1.64
192.168.1.128
192.168.1.192
```

`192.168.1.64/26` natijasi:

```text
Network address:   192.168.1.64
First usable host: 192.168.1.65
Last usable host:  192.168.1.126
Broadcast address: 192.168.1.127
Usable hosts:      62 ta
```

## Yana bir amaliy misol

Berilgan:

```text
192.168.10.77/27
```

1. `/27` mask:

```text
255.255.255.224
```

2. Block size:

```text
256 - 224 = 32
```

3. Subnetlar:

```text
192.168.10.0
192.168.10.32
192.168.10.64
192.168.10.96
192.168.10.128
192.168.10.160
192.168.10.192
192.168.10.224
```

4. `77` qaysi oraliqda?

```text
64 - 95
```

Natija:

```text
Network:   192.168.10.64
Broadcast: 192.168.10.95
Hosts:     192.168.10.65 - 192.168.10.94
```

## Network yoki broadcast addressni hostga berish

Noto'g'ri:

```text
192.168.1.0/24
192.168.1.255/24
```

`192.168.1.0/24` - network address.

`192.168.1.255/24` - broadcast address.

To'g'ri host range:

```text
192.168.1.1 - 192.168.1.254
```

## Qisqa xulosa

Subnetni tushunishda network address, broadcast address va usable host range alohida ko'riladi. Network address subnetning nomi, broadcast address subnetdagi hammaga yuborish addressi, host range esa qurilmalarga beriladigan real addresslar oralig'idir.
