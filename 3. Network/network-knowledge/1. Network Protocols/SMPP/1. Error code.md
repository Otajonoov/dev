## Qo'shimcha Ma'lumotlar

### Kategoriyalar bo'yicha tushuntirish:

**1. Asosiy Protokol Xatoliklari:**

- SMPP sessiyasining asosiy boshqaruvi bilan bog'liq
- Bog'lanish, autentifikatsiya va asosiy xabar formatiga oid xatoliklar

**2. Xabar Qayta Ishlash Xatoliklari:**

- SMS xabarlarini yuborish va qayta ishlash jarayonidagi xatoliklar
- Manzillar, xabar mazmuni va formatiga oid muammolar

**3. TLV Xatoliklari:**

- SMPP 3.4 versiyasida qo'shilgan qo'shimcha parametrlar bilan bog'liq
- Tag-Length-Value formatidagi ma'lumotlar xatoliklari

**4. Yetkazib Berish va Translyatsiya Xatoliklari:**

- SMS yetkazib berish holati va cell broadcast xizmatlariga oid xatoliklar
- Translyatsiya (broadcast) xabarlar bilan bog'liq muammolar
