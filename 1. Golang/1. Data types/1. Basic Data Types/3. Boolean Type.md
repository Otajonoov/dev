Go имеет логический тип, bool, и он настолько прост, насколько это возможно — он может быть либо true, либо false:
```go
var a bool = true  // true
var b bool         // false (значение по умолчанию - zero value)
c := !a            // false
```

Теперь, bool занимает 1 байт (8 бит) памяти. Хотя это может показаться чрезмерным — ведь булевое значение нуждается только в 1 бите для представления true или false. Каждая переменная имеет адрес памяти, и на большинстве современных систем наименьшая адресуемая единица — 1 байт. Вы не можете напрямую получить доступ к одному биту в памяти.
