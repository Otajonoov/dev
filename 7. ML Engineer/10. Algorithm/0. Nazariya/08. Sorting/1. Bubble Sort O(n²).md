
- **Tavsif**: Elementlar qo‘shni juftliklarga solishtiriladi va kerakli joyga almashtiriladi. Har bir o'tishda eng katta (yoki kichik) element orqa tomonga "ko‘chadi".
- **Kamchilik**: Katta massivlar uchun juda sekin.


### Bubble Sort (Pufakcha saralash)

Massivni boshidan oxirigacha aylanib chiqamiz, yo'l-yo'lakay tartiblanmagan qo'shni elementlarni almashtirаmiz. Birinchi o'tishda eng katta element oxirgi o'ringa "suzib chiqadi". Agar bir o'tish davomida hech qanday almashtirish bo'lmasa, massiv tartiblangan. 
Davom etib, massivning tobora qisqarib borayotgan tartiblanmagan qismini aylanib, topilgan maksimumlarni oxiriga surib boramiz. Ko'pi bilan `n` iteratsiyadan so'ng massiv tartiblanadi.

```
Eng yomon holat:  O(n²)
O'rtacha holat:   O(n²)
Eng yaxshi holat: O(n)
```

```go
func bubbleSort(arr []int) []int {
	n := len(arr)
	for i := n - 1; i > 0; i-- {
		changed := false
		for j := 0; j < i; j++ {
			if arr[j] > arr[j+1] {
				arr[j], arr[j+1] = arr[j+1], arr[j]
				changed = true
			}
		}
		if !changed {
			return arr
		}
	}
	return arr
}

func main() {
	arr := []int{5, 25, 10, 7, 6, 20, 21, 1, 2, 13}
	fmt.Println(bubbleSort(arr))
}
```
