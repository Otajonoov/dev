# Bit Manipulation (Bitli amallar)

Kompyuter sonlarni ikkilik (binary) ko'rinishda saqlaydi. **Bit manipulation** — shu bitlar ustida to'g'ridan-to'g'ri amal bajarish. Ko'p masalalarni O(1) xotira bilan, hayratlanarli darajada qisqa yechish imkonini beradi.

## Asosiy operatorlar (Go)

| Operator | Nomi | Misol | Natija |
| -------- | ---- | ----- | ------ |
| `&`  | AND | `1100 & 1010` | `1000` — ikkalasida ham 1 |
| `\|` | OR  | `1100 \| 1010` | `1110` — hech bo'lmasa birida 1 |
| `^`  | XOR | `1100 ^ 1010` | `0110` — faqat bittasida 1 |
| `&^` | AND NOT | `1100 &^ 1010` | `0100` — birinchisida 1, ikkinchisida 0 |
| `<<` | chapga surish | `1 << 3` | `1000` (= 8) |
| `>>` | o'ngga surish | `1000 >> 3` | `1` |

## Oltin formulalar

```go
n & 1          // oxirgi bit: toq/juftlikni tekshirish
n >> 1         // 2 ga bo'lish
n << 1         // 2 ga ko'paytirish
n & (n - 1)    // ENG O'NG 1-bitni o'chiradi ← eng muhim trick
n & (-n)       // eng o'ng 1-bitni ajratib oladi
1 << k         // faqat k-bit 1 bo'lgan mask
n | (1 << k)   // k-bitni 1 qilish
n &^ (1 << k)  // k-bitni 0 qilish
n ^ (1 << k)   // k-bitni teskarisiga o'zgartirish
```

### `n & (n-1)` nega ishlaydi?

`n - 1` eng o'ng 1-bitni 0 qiladi, undan o'ngdagi hamma 0 larni 1 qiladi:

```
n     = 10110100
n - 1 = 10110011
n&(n-1)=10110000   ← eng o'ng 1 yo'qoldi
```

Bundan ikkita klassik natija:
- **Power of Two**: `n > 0 && n & (n-1) == 0` — ikkining darajasida faqat bitta 1-bit bo'ladi
- **1-bitlarni sanash** (Hamming weight): necha marta `n &= n-1` qilolsang, shuncha 1-bit bor

## XOR sehri

XOR'ning xossalari: `a ^ a = 0`, `a ^ 0 = a`, kommutativ va assotsiativ.

```go
// Single Number: hamma son 2 martadan, bittasi 1 marta — o'shani top
res := 0
for _, v := range nums { res ^= v } // juftlar yo'qoladi, yolg'iz qoladi

// Missing Number ham xuddi shunday: indekslar ^ qiymatlar
```

## Reverse Bits shabloni

```go
func reverseBits(num uint32) uint32 {
    var res uint32
    for i := 0; i < 32; i++ {
        res = (res << 1) | (num & 1) // oxirgi bitni res ga qo'shamiz
        num >>= 1
    }
    return res
}
```

## Qachon ishlatasan? (signallar)

- "Hamma element **2 martadan**, bittasi 1 marta" → XOR
- "**Ikkining darajasimi**?" → `n & (n-1) == 0`
- "1-bitlar sonini sanash" → `n & (n-1)` loop
- Juda cheklangan xotira / flaglarni siqib saqlash → bit mask
- To'plamning barcha subset'larini generatsiya qilish → `0..2ⁿ-1` mask

> Bit Array (bitset) — xuddi shu g'oyaning strukturasi: har bir bit bitta boolean flag. n ta boolean o'rniga n/8 bayt xotira.
