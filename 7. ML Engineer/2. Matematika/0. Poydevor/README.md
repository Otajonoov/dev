# 0. Poydevor — noldan maktab matematikasi

**Boshlang'ich nuqta:** nol (5-sinf darajasi). Bu normal — reja aynan shunga mo'ljallangan.
**Asosiy qurol:** [uz.khanacademy.org](https://uz.khanacademy.org) (mashqlar va unit testlar bilan)
**Qo'shimcha:** [@KhanAcademyUzbek](https://www.youtube.com/@KhanAcademyUzbek) YouTube kanali — faqat takrorlash uchun (yo'lda, naushnikda). Faqat video ko'rish = o'rganish EMAS.

## Qoidalar

1. **Test-birinchi:** har bo'limni boshlashdan OLDIN uning Unit test'ini yechib ko'r. 80%+ chiqsa — bo'limni o'tkazib yubor, yiqilgan mavzularnigina o'qi.
2. **Qog'oz majburiy:** har mavzuda mashqlarni qo'lda yech. Khan Academy'ning mastery tizimi (80% qoida) shuning uchun.
3. **Har kuni 45–60 daqiqa.** Marafon emas — miya oraliqda mustahkamlaydi.
4. Bo'lim tugadi = Unit test 80%+ → checkbox'ni belgila.

> Bo'lim nomlari saytda bu yerdagidan biroz farq qilishi mumkin — mavzu bo'yicha top.

## B-bosqich: Boshlang'ich algebra (~3–4 hafta)

Kurs: <https://uz.khanacademy.org/math/pre-algebra>

- [ ] Nisbat, proporsiya va foiz
- [ ] O'zgaruvchi va algebraik ifodalar
- [ ] Bir va ikki qadamli tenglamalar
- [ ] Daraja va ildiz asoslari
- [ ] Koordinata tekisligi

## C-bosqich: Algebra I — tanlab (~6–8 hafta)

Kurs: <https://uz.khanacademy.org/math/algebra>

To'liq o'rganiladi:
- [ ] Algebra asoslari
- [ ] Tenglama va tengsizliklarni yechish
- [ ] Chiziqli tenglamalar va grafiklar ← *linear regression = shu*
- [ ] Chiziqli tenglama shakllari
- [ ] Funksiyalar ← *eng muhim bo'lim, f(x) — ML tili*
- [ ] Darajalar va ildizlar
- [ ] Eksponensial o'sish va kamayish

O'rtacha (asosiy g'oya + mashqlarning yarmi):
- [ ] Tenglamalar sistemasi ← *matritsalarga motivatsiya*
- [ ] Modul va bo'lakli funksiyalar ← *ReLU, L1 loss shu yerdan*
- [ ] Kvadrat funksiyalar va tenglamalar ← *parabola = eng sodda loss surface*
- [ ] Ketma-ketliklar ← *Σ yozuviga tayyorlov*

O'tkazib yuboriladi: Birliklar bilan ishlash, Irratsional sonlar, Ijodiy algebra.

## D-bosqich: Algebra II — tanlab (~4–6 hafta)

Kurs: <https://uz.khanacademy.org/math/algebra2>

To'liq o'rganiladi:
- [ ] **Logarifmlar** ← *ikki kursdagi ENG muhim bo'lim: cross-entropy, log-likelihood*
- [ ] Ratsional darajalar va ildizlar
- [ ] Eksponensial modellar ← *sigmoid, softmax'ning ota-onasi*
- [ ] Funksiya transformatsiyalari ← *activation funksiyalarni "ko'rish"*

Yengil (faqat g'oya):
- [ ] Ko'phadlar arifmetikasi
- [ ] Ko'phad grafiklari
- [ ] Trigonometriya — faqat sin/cos grafigi ← *positional encoding*

O'tkazib yuboriladi: Kompleks sonlar, Ko'phadlarni bo'lish/faktorlash, Modellashtirish.
