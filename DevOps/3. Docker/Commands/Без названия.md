
---
1. ==Containerga kirish:==

```shell
docker exec -it <container> <command>
```

```shell
docker exec -it postgres bash
```
---

2. ==Container ichidagi bash dan postgresga kirish:==

```shell
psql -U <user> -d <db>
```

```shell
psql -U root -d postgres
```

---



```bash
docker exec -it postgres psql -U app_user -d app_db
```

**Izoh:** Agar `-d` (database) flag'ni ko'rsatmasangiz, psql user nomidagi database'ni qidiradi. Sen `psql -U app_user` deb yozganda, u `app_user` nomli database'ni qidirdi, lekin bunday database yo'q - faqat `app_db` bor.
