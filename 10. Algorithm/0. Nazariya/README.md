# Nazariya

100 kunlik challenge'ning nazariy asosi. Har fayl bitta mavzu: tushuncha, ishlash printsipi, Go shablonlari va "qachon ishlatasan" signallari. Fayllar challenge bloklari tartibida raqamlangan — [100 kunlik reja](../1.%20Amaliyot/100-kunlik-challenge.md)dagi har blok shu yerdagi mos faylga havola qiladi.

| # | Mavzu | Asosiy g'oya |
| - | ----- | ------------ |
| 00 | [Big O](00.%20Big%20O.md) | Murakkablikni baholash, constraint'dan yechim taxmini |
| 01 | [Array va String](01.%20Array%20va%20String.md) | Indeksli kirish O(1), Go slice/string amaliyoti |
| 02 | [Two Pointers](02.%20Two%20Pointers.md) | Ikki indeks bilan O(n²) → O(n) |
| 03 | [Matrix (2D Array)](<03.%20Matrix%20(2D%20Array).md>) | Transpose, rotate, spiral, in-place tuzoqlar |

## O'qish tartibi

Challenge'ga ergashsang, tartib o'zi chiqadi: har blok boshida tegishli nazariyani o'qiysan, keyin misollarini yechasan. Mustaqil o'qisang ham raqamlar ketma-ketligi — oddiydan murakkabga qurilgan o'rganish yo'li.
