
- O'sish tartibi (ascending)**: kichikdan kattaga
- Kamayish tartibi (descending)**: kattadan kichikka

#### I. Tartiblash Algoritmlari Turlari
###### 1. Taqqoslashga Asoslangan Algoritmlar
	-Quick sort:      Pivot element yordamida bo'linadi
	-Merge sort:      Rekursiv ravishda ikkiga bo'lib birlashtiradi
	-Heap sort:       Max-heap yordamida tartiblaydi
	-Bubble sort:    Qo'shni elementlarni solishtiradi
	-Insertion sort: Elementlarni ketma-ket joylashtiradi
	-Selection sort: Har safar eng kichik elementni tanlaydi
	-Shell sort:     Insertion sortni turli intervallarda amalga oshiradi
	-Timsort:        Merge va Insertion sort kombinatsiyasi

###### 2. Taqqoslashsiz Algoritmlar
	-Counting sort: Elementlarni sanab tartiblaydi
	-Radix sort:    Raqamlarni digitlar bo'yicha tartiblaydi
    -Bucket sort:   Elementlarni diapazonlarga ajratadi


#### II. In-place Tartiblash Algoritmlari

###### 1. To'liq In-place Algoritmlar (O(1) qo'shimcha xotira)
- Bubble sort
- **Selection sort**
- **Insertion sort**
- **Shell sort**
- **Heap sort**

###### 2. Qisman In-place Algoritmlar (cheklangan qo'shimcha xotira)
	-Quick sort (O(log n) rekursiya steki uchun)

###### 3. In-place Bo'lmagan Algoritmlar
	-Merge sort     (O(n) qo'shimcha xotira)
	-Counting sort (O(n+k) xotira)
	-Radix sort    (O(n+k) xotira)
	-Bucket sort   (O(n+k) xotira)
	-Tim sort       (O(n) xotira)


#### III. Adaptiv Algoritmlar

###### 1. Qisman Tartiblangan Ma'lumotlar uchun Tezroq Ishlaydigan Algoritmlar
	-Insertion sort (O(n) eng yaxshi holatda)
	-Bubble sort    (O(n) eng yaxshi holatda)
	-Tim sort        (haqiqiy ma'lumotlar uchun optimallashtirilgan)
###### 2. Adaptiv Bo'lmagan Algoritmlar
	-Selection sort (har doim O(n²))
    -Heap sort       (har doim O(n log n))
    -Merge sort      (har doim O(n log n))

#### IV. Parallel Tartiblash Algoritmlari

###### 1. Tabiiy Ravishda Parallel Amalga Oshiriladigan Algoritmlar
	-Merge sort:
	    - Bo'linish bosqichi parallel
	    - Birlashtirish bosqichi parallel
	-Quick sort:
	    - Pivotdan keyingi bo'linish parallel
	    - Har bir bo'lak mustaqil tartiblanishi mumkin
######  2. Maxsus Parallel Algoritmlar
	-Bitonic sort        (GPU lar uchun optimallashtirilgan)
    -Sample sort         (Quick sort'ning parallel varianti)
	-Parallel radix sort (Katta ma'lumotlar to'plamlari uchun)
