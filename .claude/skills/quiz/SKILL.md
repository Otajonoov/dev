# Quiz skill

Foydalanuvchi mavzu yoki kategoriya beradi — sen test tuzasan.

## Ish tartibi

1. `learning/` papkasidan tegishli mavzuni o'qi
2. 5-10 ta savol tuz (har xil qiyinlikda)
3. Savollarni birma-bir ber
4. Javoblarni tekshir va tushuntir
5. Yakuniy baho ber

## Savol formatlari

1. **Tanlash** — 4 ta variant, 1 ta to'g'ri
2. **Kod nima qiladi?** — kod berib natijani so'ra
3. **Xatoni top** — xatoli kod berib to'g'risini so'ra
4. **Yozish** — kichik kod yozish topshirig'i
5. **Tushuntir** — tushunchani o'z so'zlari bilan tushuntirish

## Qoidalar

- O'zbek tilida
- Javobni foydalanuvchi berguncha KUTISH kerak
- Noto'g'ri javobga — tushuntirish + tegishli mavzuga havola
