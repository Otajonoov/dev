# Array va String

## Array

**Array** — bir xil turdagi elementlarni xotirada **ketma-ket** joylashgan holda saqlaydigan struktura. Har bir element o'z indeksiga ega, shuning uchun indeks bo'yicha kirish O(1).

Tasavvur qil: mehmonxona koridoridagi xonalar — har birining raqami bor, 5-xonani topish uchun barcha xonalarni tekshirish shart emas, to'g'ridan-to'g'ri borasan.

| Amal                        | Murakkablik | Izoh                              |
| --------------------------- | ----------- | --------------------------------- |
| Indeks bo'yicha o'qish      | O(1)        | `nums[i]`                         |
| Oxiriga qo'shish            | O(1)*       | `append` (amortized)              |
| Boshiga/o'rtasiga qo'shish  | O(n)        | qolganlarini surish kerak         |
| O'chirish (o'rtadan)        | O(n)        | qolganlarini surish kerak         |
| Qidirish (tartiblanmagan)   | O(n)        | linear search                     |
| Qidirish (tartiblangan)     | O(log n)    | binary search                     |

### Go'da slice

Go'da asosan `slice` ishlatiladi — dinamik o'lchamli array ustidagi "oyna":

```go
nums := []int{3, 1, 4, 1, 5}
nums = append(nums, 9)      // oxiriga qo'shish
n := len(nums)              // uzunlik
sub := nums[1:3]            // kesim: [1, 4]

// Iteratsiya
for i, v := range nums {
    fmt.Println(i, v)
}

// O'rtadan o'chirish (i-indeks) — O(n)
nums = append(nums[:i], nums[i+1:]...)
```

### Primitive turlar haqida eslatma

Array ichida saqlanadigan eng oddiy turlar — **integer**, **float**, **boolean**, **character** (Go'da `rune`/`byte`). Ular to'g'ridan-to'g'ri xotirada saqlanadi va murakkab strukturalar shulardan quriladi.

## String

**String** — belgilar (character) ketma-ketligi. Go'da string **immutable** (o'zgarmas): bir marta yaratilgach, belgini almashtirib bo'lmaydi — yangi string yaratiladi.

```go
s := "salom"
b := []byte(s)      // baytlar bilan ishlash (ASCII)
r := []rune(s)      // Unicode belgilar bilan ishlash
b[0] = 'S'
s2 := string(b)     // "Salom"

// Samarali qurish — strings.Builder (O(n))
var sb strings.Builder
for i := 0; i < 5; i++ {
    sb.WriteByte('a')
}
res := sb.String()
```

> **Muhim tuzoq:** loop ichida `s += "x"` qilish O(n²) — har safar yangi string yaratiladi. `strings.Builder` yoki `[]byte` ishlat.
