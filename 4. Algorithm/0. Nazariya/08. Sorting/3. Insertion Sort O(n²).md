- **Vaqt murakkabligi**: **O(n²)**
    
- **Xotira murakkabligi**: **O(1)**
    
- **Tavsif**: Elementlar birma-bir qo‘shiladi va tartiblangan qismga joylashtiriladi. Har safar yangi element tartiblangan massivning to‘g‘ri joyiga kiritiladi.
    
- **Afzallik**: Kichik massivlar yoki amaliyotlarda yaxshi ishlaydi (masalan, ma'lumotlar qisqa vaqt ichida o‘zgaradigan joylarda)
