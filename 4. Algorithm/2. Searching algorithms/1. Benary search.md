Бинарный поиск — это алгоритм поиска элемента в отсортированном массиве. Он работает по принципу "разделяй и властвуй", уменьшая область поиска вдвое на каждом шаге.

## **Как работает бинарный поиск?**

1. **Определяем середину массива**
    - Находим индекс середины:
	    - ==**low = 0
	    - ==*high = len(list) - 1*==
	    - ==**mid = (low + high) / 2**==
2. **Сравниваем искомый элемент с `arr[mid]`**
    - Если искомый элемент **меньше**, продолжаем поиск **в левой части** массива (`right = mid - 1`).
    - Если **больше**, продолжаем поиск **в правой части** (`left = mid + 1`).
        
3. **Повторяем шаги 1–2**, пока не найдем элемент или не останется элементов (`left > right`).

## **Сложность алгоритма**

- **Лучший случай (O(1))** — элемент находится в центре.
    
- **Средний и худший случаи (O(log n))** — массив делится пополам на каждом шаге.


```go
func searchInsert(nums []int, target int) int {
    left, right := 0, len(nums)-1  // 1. Qidiruv doirasini belgilash
    
    for left <= right {            // 2. Binary search tsikli
        mid := (left + right) / 2  // 3. O'rtadagi elementni topish
        
        if nums[mid] == target {   // 4. Agar topilsa
            return mid
        }
        
        if nums[mid] < target {   // 5. Agar target katta bo'lsa
            left = mid + 1         //    O'ng tomonga qarab qidirish
        } else {                   // 6. Agar target kichik bo'lsa
            right = mid - 1        //    Chap tomonga qarab qidirish
        }
    }
    
    return left                    // 7. Topilmasa, qo'yish joyi
}
```
