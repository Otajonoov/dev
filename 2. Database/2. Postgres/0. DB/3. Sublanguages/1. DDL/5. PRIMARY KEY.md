
 PRIMARY KEY - Asosiy Kalit
```sql
CREATE TABLE students (
    record_book numeric(5) PRIMARY KEY,
    ...
);

-- Yoki
CREATE TABLE students (
    record_book numeric(5),
    PRIMARY KEY (record_book)
);

-- Yoki
CREATE TABLE students (
    doc_ser numeric(4),
    doc_num numeric(6),
    PRIMARY KEY (doc_ser, doc_num)
);
-- Bu **BITTA** primary key, lekin u **IKKI** ustundan iborat.
-- `doc_ser` va `doc_num` **BIRGALIKDA** bitta primary key tashkil qiladi - 2 ustun = 1 composite primary key ✅
```

**PRIMARY KEY = UNIQUE + NOT NULL**

**Xususiyatlari:**

- Har bir jadvalda FAQAT BITTA PRIMARY KEY
- NULL bo'lishi mumkin emas
- Avtomatik B-tree indeks yaratiladi
- Metama'lumotlar qismi
- Boshqa jadvallar uchun identifikator
---
```sql
CREATE TABLE student (
  id INT PRIMARY KEY,
  name TEXT NOT NULL,
  passport_number TEXT UNIQUE NOT NULL
);

CREATE TABLE users (
  id SERIAL PRIMARY KEY,
  name TEXT
);
```

- Avtomatik tarzda **UNIQUE** va **NOT NULL** bo‘ladi.
- Avtomtik B-Tree index ishlatadi
