
```sql
CREATE TABLE students (
    record_book numeric(5) UNIQUE,
    ...
);

-- Yoki nom bilan
CREATE TABLE students (
    record_book numeric(5),
    CONSTRAINT unique_record_book UNIQUE (record_book)
);
```

**Muhim:** UNIQUE uchun avtomatik B-tree indeks yaratiladi.

```sql
CREATE TABLE students (
  record_book NUMERIC(5) UNIQUE
);

CREATE TABLE students (
  record_book NUMERIC(5),
  CONSTRAINT unique_record_book UNIQUE (record_book)
);
--Ikkala yozuv ham bir xil natijani beradi. Faqat ikkinchisida constraint’ga nom berilgan bo‘ladi, bu esa uni keyinchalik `ALTER TABLE` bilan o‘zgartirish yoki o‘chirishda qulaylik yaratadi.
```

- `UNIQUE` constraint biror ustun (yoki ustunlar guruhi) uchun **har bir satrdagi qiymat boshqa satrdagi qiymatdan farq qilishini** talab qiladi.
- Bu `NULL` qiymatlarga ruxsat beradi (farqli ravishda `PRIMARY KEY` esa `NULL`ga ruxsat bermaydi).
- Jadvalda **bir nechta UNIQUE constraint bo‘lishi mumkin**.
- PostgreSQL avtomatik **B-tree index** yaratadi
