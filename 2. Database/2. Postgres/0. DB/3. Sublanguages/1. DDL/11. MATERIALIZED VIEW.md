**MATERIALIZED VIEW** - bu **saqlangan natija (result)**. U ham so'rovni eslab qoladi, ham natijani saqlaydi.

```sql
CREATE MATERIALIZED VIEW routes AS
SELECT 
    flight_no,
    departure_airport,
    arrival_airport,
    aircraft_code
FROM flights
GROUP BY flight_no, departure_airport, arrival_airport, aircraft_code;
```

**VIEW vs MATERIALIZED VIEW:**

| Xususiyat            | VIEW                           | MATERIALIZED VIEW             |
| -------------------- | ------------------------------ | ----------------------------- |
| **Ma'lumot saqlash** | ❌ Yo'q (faqat so'rov)          | ✅ Ha (natija saqlangan)       |
| **Disk joyi**        | ~1 KB                          | Ko'p (natija hajmiga bog'liq) |
| **Yangilanish**      | ✅ Avtomatik                    | ❌ Qo'lda (REFRESH)            |
| **Tezlik**           | 🐌 Sekin (har safar hisoblash) | 🚀 Tez (tayyor natija)        |
| **Aktuallik**        | ✅ Doim yangi                   | ❌ Eskirishi mumkin            |
| **Foydalanish**      | Oddiy so'rovlar                | Og'ir so'rovlar               |
### Yangilash
```sql
-- Yangilash
REFRESH MATERIALIZED VIEW routes;

-- Bo'sh Yaratish
CREATE MATERIALIZED VIEW routes 
WITH NO DATA AS
SELECT ...;

-- Keyin to'ldirish
REFRESH MATERIALIZED VIEW routes;

-- Ochirish
DROP MATERIALIZED VIEW routes;
```
