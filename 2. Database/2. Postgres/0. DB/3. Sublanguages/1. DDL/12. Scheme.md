Schema - Ma'lumotlar Bazasi Sxemasi
**Schema** - bu bazadagi mantiqiy bo'lim (papka).

### Sxemalarni Ko'rish

```sql
\dn

-- Natija:
-- public
-- bookings
```

### Joriy Sxema

```sql
-- Tekshirish
SELECT current_schema;

SHOW search_path;
-- Natija: "$user", public
```

### Sxemani O'zgartirish
```sql
-- Bitta sxema
SET search_path = bookings;

-- Ko'p sxema (tartib muhim!)
SET search_path = bookings, public;
```

### Sxema bilan Ishlash
```sql
-- To'liq nom
SELECT * FROM bookings.aircrafts;

-- Qisqa nom (agar joriy sxema bookings bo'lsa)
SELECT * FROM aircrafts;
```

### Sxemada Yaratish
```sql
-- Joriy sxemada
CREATE TABLE airports (...);

-- Aniq sxemada
CREATE TABLE my_schema.airports (...);
```
