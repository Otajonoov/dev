**VIEW** - bu **saqlangan so'rov (query)**, ya'ni virtual jadval. U ma'lumot saqlamaydi, faqat so'rovni eslab qoladi. VIEW - bu faqat **so'rovni qisqartirish** yo'li!

```sql
CREATE VIEW seats_by_fare_cond AS
SELECT 
    aircraft_code,
    fare_conditions,
    count(*) AS num_seats
FROM seats
GROUP BY aircraft_code, fare_conditions
ORDER BY aircraft_code, fare_conditions;

-- Foydalanish
SELECT * FROM seats_by_fare_cond;
```

**VIEW xususiyat:** 
1. Ma'lumot saqlamaydi
2. Har safar so'rovni bajaradi.

```sql
DROP VIEW IF EXISTS seats_by_fare_cond;
```

VIEW Afzalliklari

### 1. Xavfsizlik va Ruxsatlar

```sql
-- Faqat kerakli ustunlar
CREATE VIEW student_names AS
SELECT record_book, name FROM students;

-- Faqat kerakli qatorlar
CREATE VIEW moscow_airports AS
SELECT * FROM airports WHERE city = 'Moskva';
```

### 2. Murakkab So'rovlarni Soddalashtirish
```sql
-- Murakkab so'rov
CREATE VIEW flights_v AS
SELECT 
    f.*,
    dep.airport_name AS departure_airport_name,
    arr.airport_name AS arrival_airport_name,
    timezone(dep.timezone, f.scheduled_departure) 
        AS scheduled_departure_local
FROM flights f
JOIN airports dep ON f.departure_airport = dep.airport_code
JOIN airports arr ON f.arrival_airport = arr.airport_code;

-- Oddiy foydalanish
SELECT * FROM flights_v WHERE flight_no = 'PG0405';
```
