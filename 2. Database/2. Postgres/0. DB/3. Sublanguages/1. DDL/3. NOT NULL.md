
NOT NULL Cheklovi

```sql
CREATE TABLE students (
    name text NOT NULL,  -- Bo'sh bo'lishi mumkin emas
    doc_ser numeric(4)   -- Bo'sh bo'lishi mumkin
);
```

**Muhim:**
- `NOT NULL` = `CHECK (column_name IS NOT NULL)` ga ekvivalent
- Lekin `NOT NULL` ishlatish samaraliroq
---
```sql
CREATE TABLE employees (
  id SERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  position TEXT
);
```

- `NOT NULL` — bu ustunga **bo‘sh qiymat (NULL)** kiritishni **taqiqlaydi**.
- `NOT NULL` vs `CHECK (column IS NOT NULL)`
	- Har ikki variant funksional jihatdan **bir xil**:
	- - lekin:
		`NOT NULL` — **ancha samaraliroq va tezroq ishlaydi**, chunki:

	    - PostgreSQL bu cheklovni **past darajadagi C-code darajasida** tekshiradi.

	    - `CHECK` esa **expression sifatida** ishlanadi, bu esa nisbatan sekinroq.

```sql
CHECK (column IS NOT NULL)
```
