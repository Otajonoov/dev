 
 ALTER TABLE - Jadvalni O'zgartirish

Ustun Qo'shish
```sql
ALTER TABLE aircrafts 
ADD COLUMN speed integer;
```

Ustun Qo'shish + Cheklovlar
```sql
-- 1-qadam: Cheklovsiz qo'shish
ALTER TABLE aircrafts ADD COLUMN speed integer;

-- 2-qadam: Ma'lumot kiritish
UPDATE aircrafts SET speed = 807 WHERE aircraft_code = '733';

-- 3-qadam: Cheklovlar qo'shish
ALTER TABLE aircrafts ALTER COLUMN speed SET NOT NULL;
ALTER TABLE aircrafts ADD CHECK (speed >= 300);
```

Ustun O'chirish
```sql
ALTER TABLE aircrafts DROP COLUMN speed;
```

Ustun Nomini O'zgartirish
```sql
ALTER TABLE seats 
RENAME COLUMN fare_conditions TO fare_conditions_code;
```

Cheklov Nomini O'zgartirish
```sql
ALTER TABLE seats
RENAME CONSTRAINT seats_fare_conditions_fkey
TO seats_fare_conditions_code_fkey;
```

Tashqi Kalit Qo'shish
```sql
ALTER TABLE seats
ADD FOREIGN KEY (fare_conditions_code)
    REFERENCES fare_conditions(fare_conditions_code);
```

UNIQUE Qo'shish
```sql
ALTER TABLE fare_conditions 
ADD UNIQUE (fare_conditions_name);
```
