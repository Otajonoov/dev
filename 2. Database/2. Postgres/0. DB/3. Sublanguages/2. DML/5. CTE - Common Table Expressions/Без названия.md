### Oddiy CTE

```sql
WITH ts AS (
    SELECT 
        flight_id,
        count(*) AS fact_passengers
    FROM ticket_flights
    GROUP BY flight_id
)
SELECT * FROM ts WHERE fact_passengers > 100;
```

### Recursive CTE - Rekursiv CTE

sql

```sql
WITH RECURSIVE ranges(min_sum, max_sum) AS (
    -- Anchor (boshlang'ich)
    VALUES (0, 100000)
    
    UNION ALL
    
    -- Recursive (qayta chaqiruv)
    SELECT min_sum + 100000, max_sum + 100000
    FROM ranges
    WHERE max_sum < (SELECT max(total_amount) FROM bookings)
)
SELECT * FROM ranges;
```

**Qanday Ishlaydi:**

1. VALUES (0, 100000) - boshlang'ich
2. SELECT ... FROM ranges - o'ziga qaytish
3. UNION ALL - birlashtiryapti
4. WHERE shart bajarilmaguncha davom etadi
