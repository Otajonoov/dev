## 6. Subqueries - Ichki So'rovlar

### Scalar Subquery - Skalar Ichki So'rov
```sql
-- O'rtachadan yuqori
SELECT count(*) 
FROM bookings
WHERE total_amount > (
    SELECT avg(total_amount) FROM bookings
);
```

### IN Predicate

sql

```sql
-- Krasnoyarsk zonasidagi shaharlar o'rtasida
SELECT flight_no, departure_city, arrival_city
FROM routes
WHERE departure_city IN (
    SELECT city FROM airports 
    WHERE timezone ~ 'Krasnoyarsk'
)
AND arrival_city IN (
    SELECT city FROM airports 
    WHERE timezone ~ 'Krasnoyarsk'
);
```

### EXISTS - Mavjudlik Tekshirish
```sql
-- Moskvadan marshrut yo'q shaharlar
SELECT DISTINCT city
FROM airports a
WHERE NOT EXISTS (
    SELECT 1 FROM routes r
    WHERE r.departure_city = 'Moskva'
      AND r.arrival_city = a.city
)
AND city <> 'Moskva';
```

### Subquery in SELECT

sql

```sql
SELECT 
    model,
    (SELECT count(*) FROM seats s 
     WHERE s.aircraft_code = a.aircraft_code 
       AND s.fare_conditions = 'Business') AS business,
    (SELECT count(*) FROM seats s 
     WHERE s.aircraft_code = a.aircraft_code 
       AND s.fare_conditions = 'Economy') AS economy
FROM aircrafts a;
```

### Subquery in FROM

sql

```sql
SELECT 
    s2.model,
    string_agg(
        s2.fare_conditions || ' (' || s2.num || ')', ', '
    )
FROM (
    SELECT 
        a.model,
        s.fare_conditions,
        count(*) AS num
    FROM aircrafts a
    JOIN seats s ON a.aircraft_code = s.aircraft_code
    GROUP BY a.model, s.fare_conditions
) AS s2
GROUP BY s2.model;
```
