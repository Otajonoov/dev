## DML — Data Manipulation Language

**Ma'lumotlar bilan ishlash** (o'qish, qo'shish, o'zgartirish, o'chirish)

### Asosiy DML buyruqlari (7 ta)

1. **SELECT** - ma'lumotlarni o'qish
2. **INSERT** - yangi ma'lumot qo'shish
3. **UPDATE** - mavjud ma'lumotni yangilash
4. **DELETE** - ma'lumotni o'chirish
5. **MERGE** - shartli insert/update (PostgreSQL 15+)
6. **VALUES** - qiymatlar to'plami
7. **RETURNING** - o'zgartirilgan ma'lumotlarni qaytarish

### SELECT ichida ishlatiladigan DML keywordlar (14 ta)

1. **FROM** - qaysi table/view dan
2. **WHERE** - filtrlash sharti
3. **GROUP BY** - guruhlash
4. **HAVING** - guruh uchun shart
5. **ORDER BY** - tartiblash
6. **LIMIT** - natija sonini cheklash
7. **OFFSET** - boshidan o'tkazib yuborish
8. **FETCH** - ma'lum miqdor qaytarish
9. **DISTINCT** - takrorlanmaydigan qiymatlar
10. **ALL** - barcha qiymatlar (default)
11. **AS** - alias berish
12. **INTO** - natijani boshqa joyga saqlash
13. **WITH** - CTE (Common Table Expression)
14. **RECURSIVE** - recursive CTE uchun

### JOIN va bog'lash (8 ta)

1. **JOIN** - jadvallarni birlashtirish
2. **INNER** - inner join
3. **LEFT** - left outer join
4. **RIGHT** - right outer join
5. **FULL** - full outer join
6. **CROSS** - cross join (kartezian ko'paytma)
7. **NATURAL** - natural join
8. **ON** - join sharti
9. **USING** - umumiy ustun bo'yicha join

### Mantiqiy va taqqoslash operatorlari (14 ta)

1. **AND** - mantiqiy VA
2. **OR** - mantiqiy YOKI
3. **NOT** - mantiqiy EMAS
4. **IN** - ro'yxatda borligini tekshirish
5. **BETWEEN** - oraliqda ekanligini tekshirish
6. **LIKE** - pattern matching (case-sensitive)
7. **ILIKE** - pattern matching (case-insensitive)
8. **SIMILAR TO** - SQL regex pattern
9. **EXISTS** - subquery mavjudligini tekshirish
10. **ANY** - biron-biri uchun true
11. **SOME** - ANY bilan bir xil
12. **ALL** - barchasi uchun true
13. **IS NULL** - null ekanligini tekshirish
14. **IS NOT NULL** - null emasligini tekshirish

### CASE va expressionlar (10 ta)

1. **CASE** - shartli ifoda boshlanishi
2. **WHEN** - shart
3. **THEN** - natija
4. **ELSE** - default natija
5. **END** - CASE tugashi
6. **CAST** - tip konvertatsiyasi
7. **COALESCE** - birinchi null bo'lmagan qiymat
8. **NULLIF** - shartli null qaytarish
9. **GREATEST** - eng katta qiymat
10. **LEAST** - eng kichik qiymat

### Window / Analitik funksiyalar (8 ta)

1. **OVER** - window funksiya aniqlash
2. **PARTITION BY** - ma'lumotlarni bo'laklarga ajratish
3. **ORDER BY** - window ichida tartiblash
4. **ROWS** - qator oraliqlari
5. **RANGE** - qiymat oraliqlari
6. **PRECEDING** - oldingi qatorlar
7. **FOLLOWING** - keyingi qatorlar
8. **CURRENT ROW** - joriy qator

### Qo'shimcha DML buyruqlar (3 ta)

1. **COPY** - bulk ma'lumot import/export
2. **CALL** - stored procedure chaqirish
3. **EXPLAIN** - query execution plan ko'rish

**Jami DML keywordlari: ~64 ta**
