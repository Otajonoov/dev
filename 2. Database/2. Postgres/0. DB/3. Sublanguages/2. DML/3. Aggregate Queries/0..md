
==**Aggregate Queries**== — bu guruhlangan ma'lumotlar ustida hisob-kitoblar (sum, count, avg, min, max, ...), statistik tahlillar yoki yig'indi chiqarish uchun ishlatiladi.

##### 1. Aggregate funksiyalar

|Function|Tavsif|
|---|---|
|`COUNT(*)`|Nechta satr borligini hisoblaydi|
|`COUNT(column)`|NULL bo'lmagan qiymatlarni hisoblaydi|
|`SUM(column)`|Ustundagi qiymatlar yig'indisi|
|`AVG(column)`|O'rtacha qiymat|
|`MIN(column)`|Eng kichik qiymat|
|`MAX(column)`|Eng katta qiymat|

```sql
-- Jami foydalanuvchilar soni
SELECT COUNT(*) FROM users;

-- Har bir shahar bo‘yicha o‘rtacha yosh
SELECT city, AVG(age) FROM users GROUP BY city;

-- Har bir bo‘limda nechta xodim bor
SELECT department_id, COUNT(*) FROM employees GROUP BY department_id;

```

---

##### 3. `HAVING` (guruhlangan natijalarga filter)

`HAVING` – `WHERE` ning guruhlangan natijalardagi ekvivalenti. U `GROUP BY` dan keyin ishlatiladi.

```sql
-- Faqat 100 tadan ko‘p sotilgan mahsulotlar 
SELECT product_id, SUM(quantity) AS total_qty FROM orders GROUP BY product_id HAVING SUM(quantity) > 100;
```
---

##### 4. `DISTINCT` bilan

Agar siz unice qiymatlarni hisoblamoqchi bo‘lsangiz:

```sql
-- Nechta turdagi mahsulot sotilgan 
SELECT COUNT(DISTINCT product_id) FROM orders;`
```
---

##### 5. `FILTER (WHERE ...)`

PostgreSQL’da agregat funksiyalar ichida shart qo‘yish uchun `FILTER` ishlatiladi.

```sql
`-- Umumiy miqdor va faqat 'Electronics' toifasidagi miqdor 
SELECT   SUM(quantity) AS total,   SUM(quantity) 
FILTER (WHERE category = 'Electronics') AS electronics_qty FROM products;`
```
---

##### 6. Nested Aggregates (Yig‘indining ustida yana agregatsiya)

```sql
`-- Eng ko‘p buyurtma bergan mijozni topish 
SELECT customer_id, COUNT(*) AS order_count FROM orders GROUP BY customer_id ORDER BY order_count DESC LIMIT 1;`
```
---

##### 7. `GROUPING SETS`, `ROLLUP`, `CUBE`

Bu funksiyalar ko‘p darajali guruhlash uchun ishlatiladi.
###### `GROUPING SETS`
```sql
SELECT department, gender, COUNT(*) FROM employees GROUP BY GROUPING SETS ((department), (gender));
```

###### `ROLLUP` – Ierarxik guruhlash

```sql
SELECT department, gender, COUNT(*) FROM employees GROUP BY ROLLUP(department, gender);
```

###### `CUBE` – Har xil kombinatsiyalar
```sql
SELECT department, gender, COUNT(*) FROM employees GROUP BY CUBE(department, gender);
```
---

##### 📌 8. Aggregate bilan Subquery

```sql
-- Eng ko‘p buyurtma bergan mijoz haqida ma’lumot 
SELECT * FROM customers WHERE id = (   SELECT customer_id    FROM orders    GROUP BY customer_id    ORDER BY COUNT(*) DESC    LIMIT 1 );
```
---

##### 📌 9. Window Functions bilan Agregatsiya (OVER)

Agar siz agregatsiyani barcha satrlar uchun ko‘rsatmoqchi bo‘lsangiz, `OVER()` ishlatiladi.
```sql
-- Har bir buyurtma va jami buyurtma soni 
SELECT order_id, customer_id, COUNT(*) OVER() AS total_orders FROM orders;
```
