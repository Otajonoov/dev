
## TCL — Transaction Control Language

**Transaction boshqaruvi**

### Asosiy TCL buyruqlari (5 ta)

1. **BEGIN** - transaction boshlash
2. **COMMIT** - o'zgarishlarni saqlash
3. **ROLLBACK** - o'zgarishlarni bekor qilish
4. **SAVEPOINT** - savepoint yaratish
5. **RELEASE SAVEPOINT** - savepoint ni o'chirish

### Transaction bilan ishlash (7 ta)

1. **TRANSACTION** - transaction keyword
2. **START TRANSACTION** - transaction boshlash (BEGIN bilan bir xil)
3. **ROLLBACK TO SAVEPOINT** - savepoint gacha qaytish
4. **ISOLATION LEVEL** - izolyatsiya darajasi
5. **READ COMMITTED** - o'qilgan committed ma'lumotlar
6. **REPEATABLE READ** - takrorlanadigan o'qish
7. **SERIALIZABLE** - ketma-ket bajarish

### Qo'shimcha TCL keywordlar (2 ta)

1. **DEFERRABLE** - constraint tekshiruvini kechiktirish
2. **NOT DEFERRABLE** - darhol tekshirish

**Jami TCL keywordlari: ~14 ta**
