
`psql` — bu **PostgreSQL’ga ulanib ishlash uchun mo‘ljallangan CLI (komanda satri) client dasturidir**.

#### Backslash commands "==psql==":
##### **Jadval va ma'lumotlar strukturasiga oid buyruqlar**

| Buyruq         | Ma'nosi                                                          |
| -------------- | ---------------------------------------------------------------- |
| `\?`           | **Barcha servis/metabuyruqlar** ro'yxati                         |
| `\h`           | SQL sintaksisi bo‘yicha yordam (masalan: `\h SELECT`)            |
| `\dt`          | Faqat **jadval** (table) lar ro‘yxati                            |
| `\dv`          | **View** lar ro‘yxati                                            |
| `\di`          | **Index** lar ro‘yxati                                           |
| `\ds`          | **Sequence** lar ro‘yxati                                        |
| `\d`           | **Obyekt haqida tafsilotlar**, masalan: `\d students`            |
| `\d+ students` | Jadvalning **kengaytirilgan** tavsifi (to’liq ma’lumotlar bilan) |

##### **Bazalar, jadvallar, foydalanuvchilar haqida**

|Buyruq|Ma'nosi|
|---|---|
|`\l` yoki `\list`|Bazalar ro‘yxati|
|`\c dbname`|Boshqa bazaga ulanish|
|`\conninfo`|Hozirgi ulanish haqidagi ma’lumot|
|`\dn`|Namespacelar (schemas) ro‘yxati|
|`\df`|Funksiyalar ro‘yxati|
|`\du` yoki `\dg`|Foydalanuvchilar (roles) ro‘yxati|
|`\x`|To'liq (expanded) formatda chiqish yoqish/o‘chirish|

##### **Ma’lumotlar bilan ishlash**

| Buyruq        | Ma'nosi                                                          |
| ------------- | ---------------------------------------------------------------- |
| `\q`          | `psql` dan chiqish                                               |
| `\timing`     | Har bir buyruq bajarilish vaqtini ko‘rsatish                     |
| `\i fayl.sql` | SQL faylni bajarish                                              |
| `\e`          | SQL kodni matn muharririda tahrirlash (masalan, `vim`)           |
| `\! ls`       | Tizim buyrug‘i bajarish (Linux: `ls`, Windows: `dir`, va hokazo) |

##### **Chiqarish va eksport**

|Buyruq|Ma'nosi|
|---|---|
|`\o fayl.txt`|Natijani faylga yo‘naltirish|
|`\qecho`|Konsolga satr chiqarish (skript yozishda foydali)|
|`\copy`|Mahalliy fayldan ma’lumotni import/export qilish (`COPY` SQL emas!)|


## **psql orqali db ga ulanish**
#### 1. `psql` buyrug‘i — umumiy ko‘rinish
```bash
psql [OPTION]... [DBNAME]
```
Agar parametrlar ko‘rsatilmasa, quyidagilar **default** bo‘ladi:

- `DBNAME` — foydalanuvchi nomi bilan bir xil nomli baza
    
- `-U` — tizim foydalanuvchi nomi
    
- `-h` — `localhost` (ya'ni 127.0.0.1)
    
- `-p` — port `5432`
---

#### 🔐 2. `psql -U postgres`

```bash
psql -U postgres
```

- **Ma'nosi**: `postgres` foydalanuvchisi sifatida ulanish
    
- **DB nomi** ko‘rsatilmagan => `postgres` foydalanuvchisi nomi bilan **shunga teng nomli** DB (`postgres`) izlanadi.
    
- **Qachon ishlatiladi**:
    
    - Standart `postgres` foydalanuvchisi bilan `postgres` nomli bazaga kirish uchun
        
    - Agar siz `createdb postgres` bajargan bo‘lsangiz
---

#### 🧭 3. `psql -d demo -U postgres`

```bash
psql -d demo -U postgres
```

- `-d demo` — `demo` nomli bazaga ulanadi
    
- `-U postgres` — `postgres` foydalanuvchisi sifatida
    

✅ Bu **aniq baza va foydalanuvchi** bilan ulanishdir, eng xavfsiz usul hisoblanadi.

---

#### 🌐 4. `psql -h localhost -p 5432 -U postgres -d demo`

```bash
psql -h localhost -p 5432 -U postgres -d demo
```

- `-h localhost` — 127.0.0.1 orqali TCP/IP ulanish
    
- `-p 5432` — port raqami
    
- `-U postgres` — foydalanuvchi
    
- `-d demo` — baza
    

✅ Tarmoq orqali ulanishni aniqlashtirish kerak bo‘lsa (masalan, Docker, boshqa host, SSH tunneling).

---

#### 🖥️ 5. `psql` (umuman parametrsiz)

```bash
psql
```

- Foydalanuvchi nomi: joriy OS foydalanuvchisi (`whoami`)
    
- Baza nomi: ham shu foydalanuvchi bilan bir xil
    
- Host: `localhost`
    
- Port: `5432`
    

❌ Agar OS foydalanuvchisi nomidagi baza mavjud bo‘lmasa: `psql: FATAL: database "<user>" does not exist`
