==Aрхитектуру компьютера:==

1. ==**Книги для новичков**==
	1. Код: Тайный язык информатики_ — Ч. Петцольд
	2. Феррейра Фило: **Теоретический минимум по Computer Science** (две книги, зелёная и оранжевая)
	3. Колисниченко Д.Н. - **Linux на примерах. Практика практика и только практика**
    
2. **==Для среднего уровня и выше==**
	1. Эндрю Таненбаум и Т. Остин – **Архитектура Компьютера**.
	2. Ноам Нисан и Шимон Шокен - **Архитектура Компьютерных Систем(NanToTetris)**
    
3. **==Компьютерные сети==**
	1. Э. Таененбаум и Д. Уэзеролл - **Компьютерные Сети**
4. **==Операционные системы==**
	1. Э. Таненбаум. И Х. Бос – **Современные Операционные Системы**
	2. Эви Немент: **Unix и Linux. Руководство Системного Администратора**
    
5. **"Computer Networking: A Top-Down Approach" - James Kurose, Keith Ross**
    
6. **"TCP/IP Illustrated, Volume 1" - W. Richard Stevens**
    

**"High Performance Browser Networking" - Ilya Grigorik** Web performance, HTTP/2, WebSocket, TLS/SSL kabi backend dasturchi uchun muhim mavzularni qamrab oladi. Google muhandisi tomonidan yozilgan.

**"UNIX Network Programming" - W. Richard Stevens** Socket programming, server arxitekturasi, concurrent programming kabi backend development uchun zarur bo'lgan low-level network dasturlashni o'rgatadi.

**"Building Scalable Web Sites" - Cal Henderson** Katta hajmdagi web ilovalar uchun network arxitekturasi, load balancing, caching strategiyalari haqida.

**"Designing Data-Intensive Applications" - Martin Kleppmann** Distributed systems, replication, consistency models - backend dasturchi uchun juda muhim mavzular.

**"The Phoenix Project" - Gene Kim** DevOps va network monitoring, bu amaliy jihatdan juda foydali.
